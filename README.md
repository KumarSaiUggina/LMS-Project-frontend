# Lms Project

### Project Setup Instructions

1. Clone the Project Repo

```
    git clone https://github.com/KumarSaiUggina/LMS-Project-frontend
```

2. Move into the directory

```
    cd LMS-Project-frontend
```

3. Install dependencies

```
    npm i
```

4. Run the server

```
    npm run dev
```
